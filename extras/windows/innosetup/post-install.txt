NOTE: This installer configures certain environmental variables. If you have any CMD session active during the installation, said changes will NOT be reflected and the app will not be available in opened CMD windows. To start using Mp3VoiceStamp, simply open a new CMD window and type "mp3voicestamp" + ENTER
